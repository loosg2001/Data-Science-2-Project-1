# Data-Science-2-Project-1
This repository contains the code and the full project report for a comprehensive evaluation of various regression models across three distinct real-world datasets: **Auto MPG**, **California Housing Prices**, and **Insurance Charges**. 

The goal of this project is to explore how different modeling approaches handle varying data distributions, feature interactions, and dimensionalities. We applied and evaluated multiple techniques, including:
* Standard Multiple Linear Regression
* Regularized Regression (Ridge and Lasso)
* Transformed Regression (Sqrt, Log1p, and Box-Cox)
* Second-Order Polynomial Regression (with interaction terms)

Through rigorous cross-validation and a three-stage grid search for hyperparameter tuning, this project highlights the strengths and limitations of each model across different predictive tasks.

For a detailed breakdown of our exploratory data analysis, methodology, and a cross-dataset synthesis of our results, please read Report.pdf
